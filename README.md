# Yummy API

API for the yummy app. The api has endpoints for
adding, viewing, deleting and updating recipes

## To run the project

For development

```
$ git clone git@github.com:ashish-mw/yummy-app-api.git
$ cd yummy-app-api
$ npm install
$ npm run dev
```
